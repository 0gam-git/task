# 사전과제 


## 핵심 문제해결 전략
1. Redis를 이용한다.
2. 랜덤으로 돈을 분배한다.
3. 분배된 돈을 하나씩 가져가는 구조로 설계한다.
4. 빈번한 접근 및 확인은 cache를 이용하자.
5. TTL을 이용하여 요구사항의 만료시간을 지키자.
