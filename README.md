# 사전과제 


## 핵심 문제해결 전략
1. Redis를 이용한다.
2. 랜덤으로 돈을 분배한다.
3. 분배된 돈을 하나씩 가져가는 구조로 설계한다.
4. 빈번한 접근 및 확인은 cache를 이용하자.
5. TTL을 이용하여 요구사항의 만료시간을 지키자.

## API 문서화 및 자동화 테스트
* Postman를 이용한다.(https://www.postman.com/)
* 위 링크에서 다운로드 후, "사전과제.postman_collection.json"를 import 한다.
